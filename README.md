# Repository for final project

This project is the final capstone for the **Developing AI Applications with Python and Flask** course, part of the **IBM Full Stack Software Developer Certificate**.

---

### Overview
The objective is to build a web application that leverages AI to analyze the emotional sentiment of user-provided text.

* **Frontend:** Simple interface for text input.
* **Backend:** Python & Flask web server.
* **AI Integration:** Processes text strings to identify the dominant emotion (e.g., joy, anger, sadness).

### Key Features
* **Real-time Analysis:** Users receive immediate feedback on the emotional tone of their input.
* **Flask Architecture:** Demonstrates a clean deployment of a Python-based web application.
* **RESTful API:** Handles data exchange between the client-side and the AI processing logic.
